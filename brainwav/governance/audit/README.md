# Governance Audit

This directory stores governance audit artefacts that are referenced by core policy.
Create new entries under the appropriate subfolder:

- waivers/      Approved waiver records (see AGENTS.md policy)
- reviews/      Review checklist snapshots
- incidents/    Incident reports and ESRH logs
- drills/       Emergency stop drill evidence
- agentfacts/   Agent verification samples and reports
