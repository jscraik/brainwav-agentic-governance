# Getting Started

## Prerequisites
- Node.js 20 or 22
- pnpm 10
- Docker (optional for containerized services)

## Installation
```bash
git clone https://github.com/cortex-os/cortex-os.git
cd cortex-os
./scripts/dev-setup.sh
```

## First Launch
```bash
pnpm dev
```
This boots the runtime with default configuration.
